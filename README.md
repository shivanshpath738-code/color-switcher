# 🎨 Color Switcher JS

A simple and interactive **theme/color switcher** built using **HTML, CSS, and JavaScript**.
This project allows users to dynamically change the UI theme with a single click using DOM manipulation.

## ⚙️ How It Works

* Each button has a unique class (`one`, `two`, `three`, `four`)
* When clicked:

  * JavaScript detects the class
  * Applies it to `body`, `heading`, and `footer`
* CSS styles update instantly based on the class
